- **svg 经常被用来设计一些图标，一些非常简单的可以直接手写，不过更多时候还是使用设计软件制作。**
- 在学习 svg 的过程中，可以自己找几个简单的图标来自行练习一下。
- 示例中绘制的扫码图标，使用的是 `<path>` 绘制直线的相关知识来实现的。
- 能够看懂代码就行。实现方式不唯一，做法有很多种，比如一条条地绘制 `<line>` 也行。
